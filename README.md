# 라즈베리파이 물류로봇 프로젝트  

# 설치하는 법 
1. 가상환경 설정 
```
python3 -m venv venv 
```
2. pip로 필요한 라이브러리 설치
```
pip install -r requirements.txt
```

## 라즈베리파이 4 

|GPIO#|기능|--|
|------|---|--|
|12|AIN2|오른쪽 모터를위한  PWM 신호 입력|  
|13|AIN1|왼쪽 모터를 위한 PWM 신호 입력|
|19|DIG2|오른쪽 모터의 방향 신호 입력|
|16|DIG1|왼쪽 모터의 방향 신호 입력|
